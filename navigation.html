Navigation

Links:

Navigational external links should follow these guidelines.

<a>: link

  • color: #0000ff; (blue)
  • text-decoration: underline;
  • cursor: pointer;
  • font-weight: bold; (700)

<a>: visited

  • color: #800080; (purple)
  • text-decoration: none;

<a>: hover

  • color: #ffff00; (red)
  • text-decoration: underline;

<a>: active

  • color: #ff0000; (red)
  • text-decoration: none;
    • /ul>

This is an external link example.


** Note - Links to internal pages of the website do not necessarily need to follow these guidelines! **


Buttons:

Buttons should follow these guidelines.

<.button>

  • display: inline-block;
  • border: 2px solid #b9b9b9 outset;
  • border-radius: 5px;
  • text-decoration: none;
  • cursor: pointer;
  • padding: .25rem;
  • font-weight: bold; (700)
  • color: #0000ff;

<.button>: hover

  • color: #00ff00; (magenta)
  • background-color: #000; (black)
  • border: 3px solid #b9b9b9 inset;
